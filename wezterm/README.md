# Installation

```bash
$ brew install --cask wezterm
$ cp dotfiles/wezterm/.wezterm.lua ~/
```

## Fonts

### MesloLG

```bash
$ brew install font-meslo-lg-nerd-font
```

### Powerline (old, not currently used)

```bash
$ git clone https://github.com/powerline/fonts.git
$ cd fonts
$ ./install.sh
$ cd ..
$ rm -rf fonts
```
